ABOUT

山田 新菜Niina Yamada

前職はIT業界でヘルプデスクとして従事。
公共職業訓練校テラハウスICAへ入校し、
Webデザインについて学習しました。
Webデザイナーを目指して転職活動中です。

SKILL

Design

ユーザー目線を大切にしたデザイン制作を心掛けています。
ペルソナを設定し、誰に・何を伝えるかを明確にしたうえで、 文字の配置や余白、視線誘導などこだわった設計を行い、 見た目の美しさだけでなく、ユーザーの行動につながるデザインを意識しています。

illustrator / Photoshop / Figma

Coding

デザインの意図を正確に再現できるよう、丁寧なコーディングを心がけています。
HTMLとCSSの構造を整理し、誰が見てもわかりやすく、 修正しやすい整理されたコードを書くことを大切にしています。

HTML / CSS / jQuery

Illustration

アイコンやちょっとした手描きイラストを制作することができます。 Webデザインに不足しているパーツを補完できます。サイト全体の雰囲気に合うオリジナルの表現へ仕上げます。
本サイト内で使用しているイラストも一部自分自身で制作したものです。

illustrator / procreate
